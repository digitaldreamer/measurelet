Measurelet

Boomkarklet to help measure distances and sizes.

Download this project as a .zip file Download this project as a tar.gz file

Install

Drag the button bellow into your bookmark bar.

Measurelet

Usage

  • click to place target marker
  • shift click to place source marker
  • you can drag any marker to update their place over the page
  • click hide to get access to the original web page
  • click show to show the markers and reset their position

Build from Source

Requirements

  • java
  • node
  • less
  • make

The available targets in make. The compiled and compressed files will be put into the ./build directory.

$ make build  # default
$ make css  # only make the css
$ make js  # only make the js
$ make debug  # builds without compressing the static files
$ make clean  # cleans the build directory
